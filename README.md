# Índice del Repositorio

Este repositorio contiene materiales para el curso DIW-25-26, organizados en temas y actividades.

## Estructura de Directorios

*   `tema-1/`
    *   `ejercicios/` - Contiene ejercicios para el Tema 1.
        *   `soluciones/` - Soluciones a los ejercicios.
            *   `ejercicio-1/` - Solución para el Ejercicio 1.
            *   `ejercicio-2/` - Solución para el Ejercicio 2.
            *   `ejercicio-3/` - Solución para el Ejercicio 3.
            *   `ejercicio-4/` - Solución para el Ejercicio 4.
            *   `ejercicio-5/` - Solución para el Ejercicio 5.
    *   `actividades/` - Contiene actividades para el Tema 1.
        *   `actividad-1/` - Materiales de la Actividad 1.
